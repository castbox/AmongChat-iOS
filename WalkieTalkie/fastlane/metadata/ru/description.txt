Это подходящее приложение для голосового чата, подходящее для вас, чтобы находить, сопоставлять, связываться и разговаривать с друзьями.

С этим приложением голосового чата ваш игровой опыт будет значительно улучшен! Это определенно отличное место для общения с друзьями.

Ключевая особенность
- Один щелчок, чтобы начать говорить/общаться
- Чат в реальном времени
- Создавайте свои собственные публичные или частные каналы
- Голосовой чат с друзьями на одном канале

Каналы будут контролироваться круглосуточно и без выходных в случае любого неприемлемого контента, чтобы обеспечить безопасный и чистый чат. При использовании приложения соблюдайте «Принципы сообщества».
